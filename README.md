# Password Generator (Mini Tool)

A small, client-side password generator built with HTML, CSS, and vanilla JavaScript.

## Features
- Generate passwords of configurable length
- Toggle inclusion of uppercase, lowercase, numbers, and symbols
- Strength meter and copy-to-clipboard button
- Auto-regenerates when controls change

## Usage
1. Open the site.
2. Adjust the length slider and option checkboxes.
3. Click `Generate Password` (or use the controls that auto-update).
4. Use the `Copy` button to copy the password to the clipboard.

## Files
- index.html — UI markup
- script.js — password generation, strength calculation, clipboard handling
- style.css — layout and visuals
